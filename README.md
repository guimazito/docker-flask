## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Docker
* Python
  



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/guimazito/docker-flask.git
   ```
2. Build the application
   ```sh
   docker compose build
   ```
3. Start the container
   ```sh
   docker compose up
   ```
